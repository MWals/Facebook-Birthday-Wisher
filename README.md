# Facebook-Birthday-Wisher
Facebook Birthday Wisher as the name suggests python script which will wish all the friends who have their Birthdays Today.

#Requirements
1. Pickle Library
2. Beautiful Soup Library
3. Requests Library
4. Termcolor Library

